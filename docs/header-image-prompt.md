# Header Image Prompt Seed

## Subject and Mood

Dark, technical illustration of cost data flowing through a CI/cloud pipeline.
The composition shows a horizontal pipeline diagram — stages rendered as minimal dark panels
(#0B1F17 base, #0F2A1F panel surfaces) connected by thin conduit lines. Each stage emits a
stream of abstract data particles. At the output end, a cluster of particles shifts from muted
red (#E5534B) to emerald (#2E9E6B), visualising the moment waste is caught and cost is
recovered. Emerald savings indicators — small glowing badges labelled with percentage values —
float above the conduit at key checkpoints. The overall palette is deep emerald-black with
single-accent emerald highlights; no other hues.

## Format Variants

Produce two crops from the same base composition:

- **Social card** — 1200 x 630 px. Centered pipeline with generous dark margins. Title lockup
  ("CostGuard") in the lower-left third in #E6EDEA at display weight. Savings badges prominent.
- **Hero strip** — 1440 x 400 px. Wide panoramic crop of the pipeline, title omitted (the page
  carries it). Pipeline fills the horizontal span with the emerald indicator cluster positioned
  at the right-center to draw the eye toward the call-to-action area below.

## Theme Continuity Note

The project mascot (a robot sea-captain on a patrol boat hunting floating cost/time/compute
waste, emerald accents, white background) is used for the README banner at `assets/costguard-banner.png`.
This header image is a companion technical visual, not a replacement — keep the aesthetic family
coherent by sharing the emerald accent and the "surveillance / detection" metaphor, but use an
abstract data-pipeline treatment rather than the nautical cartoon style.

## Negative Prompts

No real company logos, no real cloud-provider wordmarks or icons, no stock dollar-sign clip art,
no generic bar-chart clip art, no people or faces, no photographic stock backgrounds, no
bright-white or light backgrounds, no colours outside the emerald-black palette (avoid blue,
purple, orange, yellow), no decorative gradients unrelated to the emerald accent, no neon-glow
cyberpunk treatment.
